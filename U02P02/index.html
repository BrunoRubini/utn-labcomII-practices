Listas Una lista es una colección ordenada y mutable de elementos. Permite almacenar diferentes tipos de datos y puede modificarse. Permite agregar y eliminar elementos Puede contener elementos duplicados
Tuplas Una tupla es una colección ordenada e inmutable de elementos. Es similar a una lista, pero no se puede modificar después de su creación. Inmutable (no se puede modificar) Usada para datos que no deben cambiar
Diccionarios Un diccionario es una colección desordenada y mutable de pares clave-valor. Permite acceder rápidamente a los elementos mediante una clave única. Almacena datos como pares clave-valor Las claves son únicas e inmutables
